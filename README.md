# DHT22TFTSensor
Arduino TFT Display coupled to DHT22 Temp and Humidity Sensor

## TFT Screen
This is a 1.8 inch TFT Screen with 128 x 160 resolution V1.1. The pinout table for Arduino Mega is shown below

|TFT Pin Name|Arduino Pin|
|----|----|
|VCC|5V|
|GND|GND|
|CS|53|
|RESET|8|
|AO|9|
|SDA|51|
|SCK|52|
|LED|3.3V|
